🧍‍♂️ 彭铁林模组 (PTL Mod)
彭铁林模组 是一个基于 Minecraft Fabric (1.21+) 开发的趣味模组。它向游戏中添加了一种全新的、具有独特行为和声音的生物——彭铁林 (Peng Tie Lin)，以及相关的繁殖与掉落机制。

PTL Mod adds a unique new entity "Peng Tie Lin" to your world, featuring custom models, animations, sounds, and a unique "Poop" mechanic.

✨ 特性 (Features)
1. 新生物：彭铁林 (New Entity: Peng Tie Lin)
自然生成：你可以在主世界的平原和草原生物群系中发现野生的彭铁林。

自定义模型：拥有独特的人物模型和高清皮肤。

动画系统：具备完整的行走、手臂摆动和头部跟随动画。

声音系统：拥有独特的闲逛 (Ambient) 和受伤 (Hurt) 语音。

AI 行为：

会在世界中随机漫步。

会看向附近的玩家。

受到攻击会逃跑。

会游泳以防止淹死。

2. 独特的繁殖机制 (Breeding)
大便 (Poop)：这是本模组的核心物品。

吸引：手持“大便”可以吸引周围的彭铁林跟随你。

繁殖：给两只成年彭铁林喂食“大便”，它们会生下一只 迷你彭铁林 (Baby PTL)。

成长：幼年体体型较小，经过一段时间后会长成成年体。

3. 生产机制 (Production)
自动生产：成年彭铁林每隔一段时间（约 5~10 分钟）会随机发出声音并在脚下掉落一个“大便”物品。

你可以收集这些物品用于繁殖更多的彭铁林。
