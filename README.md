# 🧍‍♂️ 彭铁林模组 (PTL Mod)

![Fabric](https://img.shields.io/badge/Loader-Fabric-beige?style=flat&logo=fabric)
![Minecraft](https://img.shields.io/badge/Minecraft-1.21%2B-green)

**在 Minecraft 中添加生物“彭铁林”，以及独特的“大便”循环生态系统。**

> Adds the "Peng Tie Lin" entity and a unique "Poop" ecosystem to Minecraft.

## ✨ 核心特色 (Features)

### 1. 新生物：彭铁林
* **自然生成**：在平原和草原群系自动刷新。
* **独家定制**：拥有独立建模、高清皮肤以及**自定义语音**（闲逛与受伤音效）。
* **完整动画**：包含行走的腿部摆动与手臂挥动动画。

### 2. 💩 核心机制：大便 (Poop)
* **自动生产**：成年彭铁林每隔 5~10 分钟会随机掉落一个 **“大便”** 物品。
* **繁殖诱饵**：手持大便可吸引彭铁林跟随。
* **繁衍后代**：给两只彭铁林喂食大便，可繁殖出**迷你彭铁林 (Baby)**。幼体体型减半，会随时间长大。

## 🛠️ 前置需求 (Requirements)

* **Minecraft** 1.21+
* **Fabric Loader**
* **Fabric API**
